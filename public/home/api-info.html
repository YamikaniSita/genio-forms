API's

This service provides two public Programming Interfaces enabling customers to integrate the data collector to there own applications. All API's are RESTful and can be integrated with any programming language which supports RESTful requests. You need an API access code to use these APIs. If you have none register one here

User Access Control

This API allows you to control who can access your forms online, for example allow only your system users alone to access a form without sharing an access code.

How it works

You request via the API for a link for a user this link can only be used once and has an expiry time that you can set NB: The form must be secured ie have an access code to use this feature. The API simply bypasses the code.

How to use

  1. Make a request to the API

    2. Request must be made to the following url {{apiLink}} via POST method contained in the body must be the your API key and expiry date following is an example using JQuery

    		$.ajax({
			url: {{apiLink}},
			method: "POST",
			data: {
			//replace with your own data
				key: yourkey,
				expiry: 0.1 //the expiry is set in hours in this example the link expires in 6 minutes
			},
			success: function(result){
				//handle result
				}
			});

    The Request Response

    A successful request is an object in the following form

    			{response:{
				link: link, //this will be replaced by the link to use by the specific user
				expiry: 0.1, //the links expiry time counting has already started
				}
			}

    Error

    Errors that can be experienced include Form not found, Form not secured, System error.

    Example

    				{error:{
					type: "form_not_secure",
					remedy: "set_security"
					}
				}
